# p2-rwc-2027: ラグビーワールドカップ 2027 対応

## 背景

RWC 2027（2027年10〜11月、オーストラリア開催）はプロダクトの最大集客機会。
有料購読者数の KPI 達成に向けて、RWC 専用の体験を提供する。
Six Nations の延長線上に設計し、新たなデータモデルは最小限にとどめる。

## スコープ

対象（Phase RWC 準備 / 2027 Q2 開始）:
- RWC 2027 フィクスチャー・結果インポート
- プールステージ順位表
- ノックアウトブラケット表示
- 各国チーム固有のカラー・フラグ対応（Six Nations 以外の国）
- RWC 専用トップページセクション

対象外:
- ライブスコア更新（スクレイプは試合終了後のみ）
- 選手統計の詳細ページ
- 賭け・オッズ連携

## データモデル変更

### `competitions` テーブル

既存テーブルで対応。`family = 'rwc'`, `season = '2027'` で登録。

### `matches.round` の拡張解釈

| round | フェーズ |
|---|---|
| 1〜4 | Pool stage ラウンドロビン |
| 5 | Quarter-final |
| 6 | Semi-final |
| 7 | Bronze final |
| 8 | Final |

### 新規テーブル: `competition_pools`

| カラム | 型 | 説明 |
|---|---|---|
| id | uuid PK | |
| competition_id | uuid | `competitions.id` 参照 |
| pool_name | text | `'Pool A'` 〜 `'Pool D'` |
| team_id | uuid | `teams.id` 参照 |

Pool ごとの順位表は既存 `calculate-standings.ts` を pool_name でフィルタして算出。

### `teams` テーブルへの追加

RWC 出場 20 チームのうち Six Nations の 6 チーム以外を追加。
`team-identity.ts` にカラー・フラグも追加。

主な追加チーム:
- 南アフリカ（`south-africa`）、ニュージーランド（`new-zealand`）
- オーストラリア（`australia`）、アルゼンチン（`argentina`）
- その他出場国（予選確定後）

## API サーフェス

既存ルートで対応:
- `/c/rwc/2027` — RWC 2027 シーズンページ
- `/matches/[id]` — 試合詳細

新規ルート:
- `/c/rwc/2027/bracket` — ノックアウトブラケットページ

## UI サーフェス

### RWC トップページセクション

- RWC 2027 カウントダウン（開幕まで X 日）
- Pool 振り分け表
- 直近の RWC 試合カード

### ノックアウトブラケット（`components/knockout-bracket.tsx`）

- QF / SF / 決勝をビジュアル表示
- 各試合は既存 `match-card` スタイルで表示

### Pool 順位表

既存 `standings-table` を pool 別にフィルタして表示。

## コンテンツパイプライン

変更なし。既存の 4 段階パイプラインが RWC 試合にもそのまま動作する。
